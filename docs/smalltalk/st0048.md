## What: µGo 例子

```go
import "libc"
import "libc.math" => m

const Pi = 3.14
const Pi_2 = Pi * 2

type MyInt :int

global x = println(1 + 2*(3+4) + -10 + double(50))

func println() => int

func main => int {}
```
